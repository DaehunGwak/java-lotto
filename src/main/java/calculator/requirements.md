# 1단계 - 문자열 덧셈 계산기

## 기능 요구사항

- `,` `:` 가 포함된 문자열을 받으면 구분해서 숫자 리스트로 추출
- 숫자 리스트를 받으면 sum
- 기본 구분자 외 커스텀 구분자 지정 기능
    - 커스텀 구분자는 `//`, `\n` 사이에 입력
    - 문자열 앞부분에 커스텀 구분자가 위치
- 문자열 계산기에 숫자 이외의 값, 음수를 전달하는 경우 `RuntimeException` 예외를 throw


## 프로그래밍 요구사항

- 들여쓰기 depth 1단계
- 메소드크기 10라인 넘지 않게
- 메소드는 한가지 일만하기
- else 사용 X
